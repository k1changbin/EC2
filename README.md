# EC2
중간고사 대체과제 streamlit을 EC2 환경에서 실행

## 🔐 실행 방법 (로그인 및 보안 설정)

본 프로젝트는 보안을 위해 st.secrets를 사용합니다. 로컬에서 실행 시 .streamlit/secrets.toml 파일을 생성하고 아래 내용을 추가해야 로그인이 가능합니다.


```toml
[passwords]
soccer = "2026"
abcd = "1234"
```

## 선수 테이터 
FIFA26 능력치(PAC, SHO, PAS, DRI, DEF, PHY)를 기반으로  
플레이스타일 분석에 적합한 지표(공격성, 속도, 창의성, 수비력, 팀워크)로 재구성
- 창의성은 패스와 드리블 능력치를 결합하여 정의
- 팀워크는 패스와 피지컬 능력치를 결합하여 정의
``` 
attack = SHO
speed = PAC
defense = DEF
creativity = (PAS * 0.5) + (DRI * 0.5)
teamplay = (PAS * 0.7) + (PHY * 0.3) 
```

## 결과값 계산 방식
### 플레이 스차일 싱크로율
- 1순위: 코사인 유사도
  - **코사인 유사도**는 크기(실력)가 아니라, 5가지 능력치가 이루는 방향(비율·패턴)을 비교
  - 절대 점수는 다르더라도 각 능력치의 비율이 동일하다면, 플레이 스타일의 밸런스(오각형 모양)는 같다고 판단
  - 예를 들어, 100점 만점 일반인과 평균 80점대 프로 선수는 절대 점수로는 차이가 있지만, 능력치 비율이 같다면 코사인 유사도 기준에서는 유사한 플레이 스타일로 매칭
- 2순위: 절대 점수 차이
  - 코사인 유사도가 완전히 동일하여 동점이 발생할 경우 사용자의 점수와 선수 데이터의 **절대 점수 차이**를 계산하여 차이가 적은 선수를 선정

### 스타일 비중
  - 전체 5가지 능력치(공격, 속도, 창의성, 수비, 팀워크)의 점수를 모두 더한 값을 분모로 두고, 각 능력치 항목의 점수를 분자로 두어 백분율(%)을 산출
